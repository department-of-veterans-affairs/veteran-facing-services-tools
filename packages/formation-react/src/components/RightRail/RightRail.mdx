To use this component you will also need to use the <RightRailBlock />
component located inside the <RightRail /> component folder.
These sidebar blocks are passed in as children to the <RightRail />
component and they then render the individual blocks of the right rail.

### Code:

```javascript
import Sidebar from '@department-of-veterans-affairs/formation-react/RightRail';
import SidebarBlock from '@department-of-veterans-affairs/formation-react/RightRail';

const firstRightRailBlock = {
  heading: 'This is the heading',
  content: <div>This is some content</div>,
  headingLevel: 3,
};

<RightRail>
  <RightRailBlock
    heading={firstRightRailBlock.heading}
    content={firstRightRailBlock.content}
    headingLevel={firstRightRailBlock.headingLevel}
  />
</RightRail>;
```

### Rendered Component:

<div class="vads-l-col--12 medium-screen:vads-l-col--4">
  <div class="medium-screen:vads-u-padding-left--4">
    <div class="vads-u-margin-bottom--5">
      <h3 class="vads-u-margin-top--0 vads-u-padding-bottom--1p5 vads-u-border-bottom--3px vads-u-border-color--primary">
        This is the heading
      </h3>
      <div>This is some content</div>
    </div>
  </div>
</div>

### RightRailBlock Props

| prop         |      type      | required | notes                                                                                      |
| ------------ | :------------: | -------: | ------------------------------------------------------------------------------------------ |
| heading      |     string     |      yes | a string that renders as the text of the heading in the right rail block                   |
| content      | React Fragment |      yes | a React fragment that renders as the content of the right rail block                       |
| headingLevel |      int       |       no | an alternative heading level if you want the header rendered as something other than an H2 |
