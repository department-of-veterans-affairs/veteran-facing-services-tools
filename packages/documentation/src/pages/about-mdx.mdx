## About MDX

### What is MDX?
We will be using the .mdx format to create our documents.
MDX is a format that lets you seamlessly use JSX in your Markdown documents.

#### Inline components
You can use components inline inside your MDX files. Below is an example
of how to do it.

- You will need to export it for it to work.
- You can write both Function components and Class components.
- The reason why you need to use this is when you need the parent state to change.

This example uses state to toggle the drop down.

```jsx
// Class Component
export class InlineClassComponent extends React.Component {
  constructor() {
    super()
    this.state = {
      open: false
    }
  }
  render() {
    return (
      <div className="site-c-reactcomp__rendered">
        <div style={{ backgroundColor: '#112e51' }}>
          <DropDownPanel
            buttonText='Helpdesk'
            cssClass='va-dropdown'
            isOpen={this.state.open}
            contents='Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce ullamcorper at eros eu suscipit. Ut imperdiet libero et luctus pretium.'
            clickHandler={() => this.setState({ open: !this.state.open }) }>
            'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Fusce ullamcorper at eros eu suscipit. Ut imperdiet libero et luctus pretium.'
          </DropDownPanel>
        </div>
      </div>
    )
  }
}

<InlineClassComponent />

// Function Component
export const InlinePresentationalComponent = () => (
  <h1>Hello World</h1>
)

<InlinePresentationalComponent />
```

***Gotchas***
- MDX doesn't like newlines in between methods in a class based component.
- Inline components need to begin with an uppercase letter. If it starts with a
  lowercase it will not render.

Bad:

```
export class inlineClassComponent extends React.Component {
  constructor() {
    super()
  }

  render() {
    return (
      <div className="site-c-reactcomp__rendered">
        Hello World
      </div>
    )
  }
}
```

Good:
```
export class InlineClassComponent extends React.Component {
  constructor() {
    super()
  }
  render() {
    return (
      <div className="site-c-reactcomp__rendered">
        Hello World
      </div>
    )
  }
}
```
